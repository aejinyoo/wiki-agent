# 파이프라인 보강 요구사항

현재 파이프라인이 놓치는 두 가지를 메운다.

## 1. 한국어 요지

- 원문(영어·일어 등)이 그대로 저장돼 읽기 부담이 큼.
- 각 아이템의 frontmatter 에 한국어 paraphrase 필드(`body_ko`) 추가.
- 단순 번역이 아닌 300~600자 요지 재정리. 원문이 이미 한국어면 비워둠.
- 본문은 `## 한국어 요지` + `## 원문 발췌` 두 섹션.

## 2. 인사이트 구조화

`summary_3lines` 한 덩어리 대신 세 필드로 분리:

- **`key_takeaways`** — 3~5 불릿, 수치·방법론 포함된 구체적 교훈.
- **`why_it_matters`** — 개인화 컨텍스트 기준 "왜 나에게 의미 있는가" (1~2문장).
- **`what_to_try`** — 동사로 시작하는 30분~3시간 단위 실행 액션 (1~2문장).
